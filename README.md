# Emotion Detection AI Web Application

An AI-powered web application that performs emotion detection on user-provided text, built using the Watson NLP Library and Flask.

## Project Overview

Emotion detection extends sentiment analysis by identifying fine-grained emotions — anger, disgust, fear, joy, and sadness — from text input. This project was developed as a final project for IBM's *Python for AI & Development Project* course.

## Features

- Emotion analysis via IBM Watson NLP Library
- Displays scores for all five emotions
- Identifies the dominant emotion
- Error handling for blank input
- Flask web interface
- Unit tested with Python `unittest`
- Passes static code analysis with PyLint (score: 10/10)

## Tech Stack

- Python 3
- Flask
- Watson NLP Library
- IBM Cloud IDE
- PyLint
- Unittest
